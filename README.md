## Rangkuman Materi Praktik
Praktik ini mencakup implementasi logika tingkat lanjut pada pemrograman Python, antara lain:

1. **Python Classes and Objects (OOP)**: 
   - Mempraktikkan pembuatan `Class` sebagai blueprint untuk membuat objek.
   - Mengimplementasikan fungsi konstruktor `__init__()` untuk inisialisasi properti objek secara otomatis.
   - Memahami penggunaan parameter `self` sebagai referensi instance class dan pembuatan `Methods` (fungsi di dalam class).
   - Demonstrasi modifikasi properti, penghapusan properti (`del`), serta penghapusan objek.

2. **Inheritance (Pewarisan)**: 
   - Membuat `Child Class` (Student) yang mewarisi sifat dan metode dari `Parent Class` (Person).
   - Menggunakan fungsi `super()` untuk memanggil fungsionalitas class induk tanpa menuliskan nama class induk secara eksplisit.
   - Implementasi `Multi Class Inheritance` untuk menggabungkan properti dari beberapa parent class sekaligus (contoh: Class Person dan Class Address).

3. **Scope & Modules**: 
   - **Scope**: Memahami batasan akses variabel melalui `Local Scope` (di dalam fungsi) dan `Global Scope` (di luar fungsi), serta penggunaan keyword `global` untuk mengubah variabel lokal menjadi global.
   - **Modules**: Praktik mengimpor modul eksternal (`import`), penggunaan alias (`as`), serta pemanfaatan built-in module seperti library `os` untuk operasi sistem file (cek direktori, cek keberadaan file, dan membuat folder).
